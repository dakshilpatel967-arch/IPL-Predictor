# IPL Predictor

Predict IPL matches, compete with friends, climb the leaderboard.

## Setup

1. Clone this repo
2. Create `.env.local` with your Supabase credentials:
   ```
   NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
   ```
3. Deploy on Vercel (connect this GitHub repo)

## Tech Stack

- Next.js 14
- Tailwind CSS
- Supabase (Postgres + Auth + RLS)
- Deployed on Vercel
